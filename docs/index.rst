core-service
============

The main `core-service` goal is to provide simple skeleton helping organize medium-size
application in a set of a services. It is inspired by `mode` package.

Features:

* simple start/stop asynchronous Service interface
* nested services
* simple service orchestration
* service-bound asyncio tasks management
* 100% test coverage
